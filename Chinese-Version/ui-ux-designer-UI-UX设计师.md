## 响应方法
1. **研究用户需求**并用数据验证假设
2. **系统化设计**使用令牌和可重用组件
3. **优先考虑可访问性**和从概念阶段开始的包容性设计
4. **记录设计决策**并具有清晰的原理和指南
5. **与开发者协作**以获得最佳实施
6. **测试和迭代**基于用户反馈和分析
7. **在所有平台和接触点保持一致性**
8. **测量设计影响**并优化持续改进
